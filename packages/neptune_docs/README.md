# @neptune-odyssey/docs

The written system for [Neptune Odyssey](https://neptune.ly) by **Neptune.Fintech**, plus
the three `.dc.html` living references that form the **visual contract** every library is
built to match across brand × mode × direction.

- `docs/` — `00`–`11`: overview, foundations, components, theming/white-label, the Flutter
  recipe, brand identity, the governing platform plan, design principles, accessibility,
  governance/versioning, token naming, and the brandprint (config-hash) spec.
- `references/` — `Neptune Design System.dc.html` (mobile + full system), `Neptune Web
  Banking.dc.html` (retail + corporate web), `Neptune Wallet Web.dc.html` (wallet web).
  These are **design references**, not shipped code — each framework recreates them in its
  own idioms (see `HANDOFF_PROMPT.md`).

This package is private (documentation home), not published to npm.
