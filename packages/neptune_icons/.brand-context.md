# Libyan payment-rail brand identities (for the placeholder marks)

The `placeholder` marks for the local rails must reflect the **specific Libyan
brands**, not generic same-named services elsewhere:

- **sadad** — the **Libyan Sadad by Almadar (المدار الجديد / Almadar Aljadid)**, the
  Libyan mobile operator's payment service. NOT Saudi SADAD / generic. Match Almadar's
  identity (green-led brand), with the Arabic **سداد** as the hero.
- **onepay** — the **Libyan OnePay (وان باي)** specifically, not any other "OnePay".
- **lypay** — Libyan LYPay / لي باي (LibyanSpider), green→blue swoosh.
- **moamalat** — Moamalat Financial Services (Libya), gold flowing "M".
- **numo** — NUMO, Moamalat's national card scheme (navy + gold).
- **tadawul** — the Libyan trading/exchange context.

All remain ORIGINAL simplified placeholders (no traced official artwork); official
SVGs drop in via `registerBrandMark()`. Each carries its native Arabic wordmark.
